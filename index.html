配信アーカイブ
⚙️ 初回設定：3箇所だけ書き換えてください
このファイルをテキストエディタで開き、下記3行を書き換えると動画が自動表示されます：

const YT_API_KEY = 'ここにAPIキーを貼る';
const YT_MAIN_CH_ID = 'メインチャンネルIDを貼る';
const YT_SUB_CH_ID = 'サブチャンネルIDを貼る';

YouTube Data API v3 キーの取得手順：
  1. Google Cloud Console でプロジェクトを作成
  2. 「APIとサービス」→「ライブラリ」→「YouTube Data API v3」を有効化
  3. 「認証情報」→「APIキーを作成」でキーをコピーして上の行に貼り付け
  4. チャンネルIDは各チャンネルの「設定 → 詳細設定」で確認できます
